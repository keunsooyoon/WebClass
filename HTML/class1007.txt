웹 개발이란...
포트폴리오 사이트나 기업 홍보사이트처럼 방문자에게 정보를 보여주는 웹 사이트는 간단하다. 이런 사이트는 정적 사이트 static site 라고 한다. 
그러나 웹 개발은 보여주는게 전부가 아니다. 예를 들면  페이스북 같은 경우 '친구 추가', '좋아요','공유하기' 등등의 서비스나  쇼핑몰 사이트의 경우도 제품을 보여만 주는게 아니라 카드결제 기능을 구현해야한다. 이러한 사이트를  동적 사이트 dynamic site 라고 한다. 즉 웹개발은 사용자들에게 제공할 기능과 서비스까지 모두 준비해야 한다.  


서버와 클라이언트 

사용자가 웹사이트에 접근하려고 사용하는 PC 타블렛 스마트폰 등을 클라이언트라고 한다. 좁은 의미로는 웹브라우저를 클라이언트라고 한다. 

클라이언트로 접근해서 정보를 가져와서 보여줄수 있도록 제공하는 컴퓨터를 서버라고 한다. 


프론트앤드 백앤드
사용자 앞에 보이는 영역을 프론트앤드, 즉 브라우저의 화면에 보이는 것들을 다룬다. 
HTML, CSS, JS

사용자의 뒤에서 보이지 않는 영역을 뜻한다. 회원 정보, 게시물, 사진들 데이터 베이스에 저장된다. 데이터베이스를 설계하거나 데이터를 처리하는 것인 백앤드 개발 영역이다. 
JAVA, PHP, ASP, 파이썬(장고, 플라스크),JS



웹개발 로드맵



기본 : HTML, CSS, JS

프론트앤드 : 라이브러리 j-query, 부트스트랩
            프레임워크 react....

백앤드 : 언어 - 자바, 파이썬, PHP
        프레임워크 - node js, 스프링, 장고...

        *프레임워크는 프로그램을 만들 때 필요한 여러가지 기능을 쉽게 구현할 수 있도록 도와주는 도구다. 예를 들면, 로그인 기능, 회원가입, 댓글기능 ....개발자가 웹사이트를 쉽게 만들 수 있도록 도와준다. 

웹개발을 하려면 브라우저에 정보를 어떻게 표현하는지 알아야한다. 그래서 프론트앤드 개발을 하든 백앤드 개발을 하든 HTML, CSS, JS 먼저 공부를 해야한다.
